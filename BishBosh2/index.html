Bish-Bosh 2.0

Explanation

Bish-Bosh is a counting game where you count out numbers in order:

  • If the number is divisible by some number x you say "Bish"
  • If the number is divisible by some number y you say "Bosh"
  • If the number is divisible by both x and y you say "Bish-Bosh"
  • Otherwise you say the number
